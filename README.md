# CryptoPair (Обзор курса криптовалютных пар)
## Description
* Обзор актуального курса различных криптовалют
* Парсинг курса криптовалют из [https://www.coinapi.io](https://www.coinapi.io)
* Выбор криптовалютной пары
## Architecture
* MVC
## Used technologies
* ParseJSON
* URLSession
* Without storyboard
